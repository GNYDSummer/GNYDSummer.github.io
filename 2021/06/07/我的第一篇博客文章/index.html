对于Win下的git cmd无法hexo init的情况：

解决办法：替换GitHub链接

hexo 的npm模块在Windows环境下是在：npm\node_modules\hexo-cli\lib\console

在这个文件夹下有一个init.js文件，打开文件

将
const GIT_REPO_URL = ‘https://github.com/hexojs/hexo-starter.git';
替换为：
const GIT_REPO_URL = ‘https://github.com.cnpmjs.org/hexojs/hexo-starter.git';

再次运行hexo init，成功！
